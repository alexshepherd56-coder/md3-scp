How This Works

Event-Driven Architecture:
  • Modules communicate through eventBus instead of calling each other directly
  • No tight coupling - modules can be added/removed without breaking others
  • Easy to test - can emit fake events to test module responses
  • Easy to debug - can see all events in the log below

üíª Try These in Browser Console (F12)

Open your browser console and try these commands: 
// See all events that have been emitted
eventBus.getHistory();

// Emit a custom event
eventBus.emit('test:event', { message: 'Hello from console!' });

// Listen to events
eventBus.on('test:event', (data) => console.log('Got event:', data));

// See all registered events
eventBus.getEvents();
